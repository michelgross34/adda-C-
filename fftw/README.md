## Install FFTW on Windows

- Download FFTW package  here 

    - https://www.fftw.org/install/windows.html

- extract fftw-3.3.5-dll64.zip (3.1MB) package  in ./fftw  

- Extract the import library and copy it to project's root directory 

    - build_libfftw.bat
    - copy libfftw3-3.dll.a ..

- copy libfftw3-3.dll in the final *.exe directory
